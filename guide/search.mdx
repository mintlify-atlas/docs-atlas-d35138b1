---
title: "Search"
description: "Full-text search across all campaign data including beacons, hosts, commands, and tags"
---

Search gives you instant access to any piece of data in the campaign. You can search across beacons, hosts, operators, commands, comments, and tags from anywhere in the application.

## Opening search

Search is available from the eyeglass icon in the navigation bar. You can also open it from anywhere within a campaign using a keyboard shortcut:

<Tabs>
  <Tab title="Windows">
    Press `Ctrl` + `K`
  </Tab>
  <Tab title="Mac">
    Press `Cmd` + `K`
  </Tab>
  <Tab title="Linux">
    Press `Ctrl` + `K`
  </Tab>
</Tabs>

## What you can search

Search runs full-text queries across eight data types:

| Data type | What it searches |
|-----------|------------------|
| **Beacons** | Beacon names and display names |
| **Hosts** | Host names and display names |
| **Teamservers** | Team server names |
| **Operators** | Operator usernames |
| **Comments** | Comment text and tag names |
| **Commands** | Command input text and output |
| **Command Type** | Command type names |
| **Tags** | Tag labels attached to comments |

## Filtering results

After entering a search query, you can narrow the results by data type. Use the filter controls above the results list to show only a specific category — for example, show only **Comments** matching your query, or only **Hosts**.

## Sorting results

Results can be sorted three ways, each supporting ascending and descending order:

| Sort option | Description |
|-------------|-------------|
| **Relevance** | Orders results by how closely they match the search query |
| **Name** | Alphabetical order by the item's name or display name |
| **Type** | Groups results by data type |

Toggle between ascending and descending for any sort option using the direction control next to the sort selector.

## Tips for effective searching

- Search for a **tag name** to quickly find all comments in a specific category before switching to Presentations
- Search for a **host name** to jump directly to that host's detail view without scrolling the Explorer
- Search for **command text** (e.g., `mimikatz`, `shell whoami`) to locate specific technique usage across the entire campaign
- Use **Operator** search to scope activity to a specific team member during an investigation

<Tip>
  The keyboard shortcut `Ctrl+K` (or `Cmd+K` on Mac) works from any screen within the campaign, making it the fastest way to navigate to a specific item without leaving your current view.
</Tip>
